# Pico 2 W Display Profile

Placeholder for the Pico 2 W display definition.

The intended first display is the original Pebble Sharp Memory LCD:

- 144 x 168
- 1-bit black/white
- rectangular Pebble display geometry

This folder should only contain Pico 2 W display wiring/profile notes and future
display headers. Shared Sharp LCD driver code should stay in the common driver
area rather than being duplicated here.
